\# Windows Desktop Control for OpenClaw



这是一个为 OpenClaw 开发的 Windows 原生桌面控制插件。



\### 功能

\- \*\*截图\*\*：获取当前屏幕快照并保存。

\- \*\*点击\*\*：模拟鼠标左键点击指定坐标。

\- \*\*移动\*\*：将鼠标光标移动到指定位置。



\### 安装方法

1\. 将本文件夹克隆或下载到 `C:\\Users\\YourName\\.openclaw\\skills\\` 目录下。

2\. 以管理员身份运行 PowerShell，并执行：

&#x20;  `Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser`



\### 使用环境

\- Windows 系统

\- PowerShell

\- OpenClaw Gateway
